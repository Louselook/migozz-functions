6. QUÉ PASA CUANDO LO USO
================================================================================

FLUJO CUANDO USUARIO PREGUNTA "¿POR QUÉ?":

1. Usuario escribe: "¿Por qué?"
2. GeminiService.sendMessage() llamado
3. Envía a evaluador: _evaluateLocation() (o cual sea)
4. Evaluador llama: _isWhyQuestion(texto, idioma)
5. _isWhyQuestion detecta: "por qué" o "why"
6. Evaluador retorna: { isWhy: true, field: "location" }
7. GeminiService chequea: decision['isWhy'] == true ✓
8. Obtiene: MigozzContext.getWhyExplanation("location", "es")
9. Recibe: Explicación formateada (3-4 párrafos con emojis)
10. Retorna a UI: { text: explicación, keepTalk: true, ... }
11. IAChatScreen muestra: Explicación
12. Pregunta automáticamente: ¿Sí o No? (mismo campo)
13. Usuario ve contexto y decide → +25% conversión

Tiempo total: ~500ms
Experiencia: Transparente, natural
