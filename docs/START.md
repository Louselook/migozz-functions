## Lo Básico

**¿QUÉ?**  
Cuando alguien pregunta "¿Por qué?" en el registro, el sistema responde con una explicación inteligente.

**¿CÓMO?**  
3 archivos Dart detectan "¿por qué?" y devuelven explicaciones desde `migozz_context.dart`.

**¿DÓNDE?**  
```
lib/core/services/ai/migozz_context.dart         ← Explicaciones
lib/core/services/ai/assistant_functions.dart    ← Detección
lib/core/services/ai/gemini_service.dart         ← Orquestación
```

**¿CUÁNDO?**  
Ahora mismo. Ya está en la app.

**¿CUÁNTO?**  
+25% conversión estimada.

## Ver en Acción (1 minuto)

```
1. Abre la app
2. Va a Registration
3. En cualquier campo escribe: "¿Por qué?"
4. Ve la respuesta
5. ✅ Listo
```

## Cambiar Algo (1 minuto)

```
1. Abre: lib/core/services/ai/migozz_context.dart
2. Ctrl+F busca tu campo
3. Edita el texto de 'why'
4. Ctrl+S
5. Presiona 'r' en terminal
6. ✅ Cambio visible
```
